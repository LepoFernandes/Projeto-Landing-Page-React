# 🚀 Projeto Landing Page React

## 🚀 Sobre o projeto

Landing page moderna desenvolvida com React, Vite e Tailwind CSS, criada com foco em design responsivo, desempenho e experiência do usuário.

O projeto foi desenvolvido para praticar conceitos importantes do ecossistema React, incluindo componentização, organização de código e criação de interfaces modernas utilizando Tailwind CSS.

## ⚙️ Como rodar localmente

Clone o repositório:

```bash
git clone https://github.com/LepoFernandes/Projeto-Landing-Page-React.git
```

Acesse a pasta do projeto:

```bash
cd Projeto-Landing-Page-React
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

O projeto estará disponível em:

```bash
http://localhost:5173
```
